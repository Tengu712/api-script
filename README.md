# api-script

## 概要

Win32APIやVulkan等の「API」は、ユーザに対し機械的な呼び出しを要求する。

この機械的な呼び出しを簡潔なスクリプトで記し、WindowsあるいはLinux向けの静的リンクライブラリに変換することを目的とする。

## コンパイラオプション

| オプション | 意味 |
| ----- | ----- |
| -p | プロセス内容を逐次プリントする。 |
| -c | ビルドせず一時ファイルのみを出力する。 |
| -d | 静的リンクライブラリとヘッダーを出力する。 |
| -a | 一時ファイルをすべて残す。 |

## 開発環境

* Ubuntu 22/20.04 LTS
* flex 2.6.4
* bison 3.5.1
